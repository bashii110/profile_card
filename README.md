# Profile Card

A simple and elegant Flutter application that displays a personal/professional profile card — perfect for portfolios, resumes, or "about me" screens.

## 📱 Preview


<img width="490" height="732" alt="Screenshot 2026-07-21 232352" src="https://github.com/user-attachments/assets/62a85556-78e8-47e9-a402-0ac4e69462d6" />



The app shows:
- A circular profile photo with a colored border
- Full name
- Contact details (email and phone number, each with an icon)
- A row of skill/tech tags (chips)
- Quick-access icons for links such as code/portfolio, work/experience, and email
- An edit button in the app bar to update profile info

## ✨ Features

- Clean, card-based UI with rounded corners and subtle elevation/shadow
- Circular avatar image display
- Contact info with `email` and `phone` icons
- Dynamic skill chips (e.g. Flutter, Dart, Firebase, Git, GitHub)
- Bottom icon row for quick actions/links
- Edit icon in the app bar for updating profile details
- Responsive layout that adapts to different screen sizes

## 🛠️ Built With

- [Flutter](https://flutter.dev) — UI toolkit for building natively compiled apps
- [Dart](https://dart.dev) — Programming language
- Material Design widgets

## 🚀 Getting Started

### Prerequisites

- [Flutter SDK](https://docs.flutter.dev/get-started/install) installed on your machine
- A code editor (VS Code, Android Studio, etc.)
- An emulator/simulator or a physical device

### Installation

1. Clone the repository
   ```bash
   git clone https://github.com/your-username/profile-card.git
   cd profile-card
   ```

2. Install dependencies
   ```bash
   flutter pub get
   ```

3. Run the app
   ```bash
   flutter run
   ```

## 🎨 Customization

To customize the profile card for your own use:

1. Replace the profile image in `assets/images/` with your own photo.
2. Update the name, email, and phone number fields in the relevant widget/screen file.
3. Edit the list of skill chips to reflect your own tech stack.
4. Update the bottom icon links (code, work, email) to point to your own portfolio, resume, or contact links.
